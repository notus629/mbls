#### 命名空间

命名空间（或子命名空间）的作用是封装和组织相关的 PHP 类。

PHP 解释器会将其作为前缀添加到类、接口、函数和常量的名称前面。

##### 声明命名空间

声明命名空间的代码始终应该放在 `<?php` 标签后的第一行。

```php
<?php
namespace Symfony\Component\HttpFoundation;
```

##### 导入
