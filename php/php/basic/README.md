#### 常量

名称 | 含义
--- | ---
PHP_EOL      | 与系统相关的换行符(`\r\n` for Win, `\n` for Mac & Linux)
PHP_INT_SIZE | 整型的大小
PHP_INT_MAX  | 整型最大大小
PHP_INT_MIN  | 整型最小值（PHP7.0)
